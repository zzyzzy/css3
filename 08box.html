CSS 박스 모델

개요

모든 html요소는 박스로 구성되며 이를 박스모델이라 함

박스모델은 content, padding, border, margin으로 구성

웹 디자인의 핵심은 컨텐츠를 담을 박스모델(블록/인라인/...)을 정의하고 CSS속성으로 스타일과 위치, 정렬을 수행하는 것임

박스모델을 통해 텍스트,이미지,테이블등의 요소를 배치할 수 있고, 이것을 통해 웹페이지의 레이아웃을 구성할 수 있음

너비/높이

width, height 속성으로 요소의 너비와 높이를 설정

단, block 요소에 대해서만 설정 가능

지정한 요소의 크기보다 내용이 많으면 요소 밖으로 벗어나 넘칠수도 있음 (overflow 속성을 이용하면 넘친 내용을 감출수 있음)

여백

margin과 padding으로 여백을 설정할 수 있음

박스 내용을 기준으로 4방향(위/오른쪽/아래/왼쪽)으로 설정 가능
